# 🏠 Real Estate Website

A simple and responsive real estate website built using **HTML, CSS, and JavaScript**.
This project allows users to browse properties and filter them based on their preferences.

---

## 🚀 Features

* 🏡 Property listing page
* 🔍 Filter properties (price, location, type)
* 📱 Responsive design (mobile + desktop)
* ⚡ Fast and lightweight (no frameworks)
* 🖼️ Property images and details

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
real-estate-website/
│
├── index.html

└── README.md
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
